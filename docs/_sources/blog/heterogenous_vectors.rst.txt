Heterogenous Vectors in C++
###########################

STL Vectors can only store a single type of object which the compiler must know
at compile time so it can allocate the right amount of memory for it. This
limitation can be circumvented with an `extra level of abstraction
<https://wiki.c2.com/?OneMoreLevelOfIndirection>`_ over the type that
std::vector takes.
